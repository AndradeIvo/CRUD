# CRUD Management System

Desktop CRUD application developed in C# and Windows Forms with MySQL database integration.

The project was created to demonstrate the implementation of fundamental database operations (Create, Read, Update and Delete) through a graphical user interface, following a structured approach to data management and validation.

## Features

* Create new records
* View existing records
* Update record information
* Delete records
* MySQL database integration
* Data validation
* User-friendly Windows Forms interface

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* MySQL
* MySQL.Data Connector

## Project Structure

The application follows a simple architecture that separates:

* User Interface (Windows Forms)
* Database Operations
* Data Validation
* Business Logic

## Database Operations

The system implements the four fundamental CRUD operations:

### Create

Insert new records into the database.

### Read

Retrieve and display stored records.

### Update

Modify existing information.

### Delete

Remove records from the database.

## Build and Run

### Requirements

* Visual Studio
* .NET Framework
* MySQL Server
* MySQL.Data Package

### Setup

1. Clone the repository

```bash
git clone https://github.com/AndradeIvo/CRUD.git
```

2. Configure the database connection string.

3. Create the required database and tables.

4. Open the solution in Visual Studio.

5. Build and run the application.
